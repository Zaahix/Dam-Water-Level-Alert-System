# Dam Water Level Alert System

## Project Overview

IoT-based Dam Water Level Alert System using ESP8266 NodeMCU and HC-SR04 Ultrasonic Sensor.

## Features

* Real-time water level monitoring
* Safe, Warning and Danger alerts
* ThingSpeak cloud integration
* LED status indication
* Water level percentage calculation

## Components

* ESP8266 NodeMCU
* HC-SR04 Ultrasonic Sensor
* LEDs
* 2.2 kΩ Resistors
* Breadboard

## Software

* PlatformIO
* ThingSpeak
* EasyEDA
* Fritzing
